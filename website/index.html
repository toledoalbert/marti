API Calls


Search for Keywords and Location

/bagofwords/geocodes

This API call allows you to pass in two parameters, a set of words you want to search for in a Tweet, as well as the geocode of a specific location you want to retrieve the Tweets from.
The bag of words should be inserted with a + as the delimeter.

Example:

http://martiapi.herokuapp.com/hampton+university/33.448260,-112.075774,100mi


Search for Keywords

/bagofwords

This API call allows you to pass in one parameters, a set of words you want to search for in a Tweet.
The bag of words should be inserted with a + as the delimeter.

Example:

http://martiapi.herokuapp.com/hampton+university
