Marti

Marti is a suite of software products for both developers and end users. Marti provides Tweet analyzing functionality to its users. Marti Search is an advanced search product for Twitter while Marti Weather provides weather information from Twitter users for the continental United States. In addition to our end user products, Marti API and Marti JavaScript SDK allow third party developers to integrate Marti functionality in their applications.
