M a r t ı

Martı is a suite of software products for both developers and end users. Martı provides Tweet analyzing functionality to its users. Martı Search is an advanced search product for Twitter while Martı Weather provides weather information from Twitter users for the continental United States. In addition to our end user products, Martı API and Martı JavaScript SDK allow third party developers to integrate Martı functionality in their applications.

Search
Martı Search is a tweet search engine for both keywords and regular expressions.

Weather
Martı Weather is a simple weather map application that retrieves the weather information from tweets across the U.S. using Martı technology.

API
Martı API is a REST API that allows third party developers to benefit from the Tweet Analyzing functionality of Martı. Using this API, developers can build third party apps with Tweet Analyzing functionality including iOS, Android, Windows Phone and Web Applications and more.

JavaScript SDK
Martı also provides a JavaScript SDK for developers who want to build desktop or mobile web applications. The SDK extends the API and adds a level of abstraction that will make the Martı functionality work seamlessly in you JavaScript workflow.
